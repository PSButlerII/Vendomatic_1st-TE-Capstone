## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

The Vendo-Matic 800 Vending Machine Software is an advanced application developed in Java, designed to enhance the vending machine experience. It is integrated with bank accounts, allowing customers to purchase products from their computers conveniently. The application manages inventory, processes transactions, and provides a user-friendly interface.

### Built With

* [Java](https://www.java.com/en/)
* [Unit Testing Tools]()
